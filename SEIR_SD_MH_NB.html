SEIR Model with Social Distancing on Maharashta Covid-19 Data

Parts of code taken from Tim Churches blog on “Analysing COVID-19 (2019-nCoV) outbreak data with R” https://timchurches.github.io/blog/posts/2020-02-18-analysing-covid-19-2019-ncov-outbreak-data-with-r-part-1/

Maharashtra Data from: https://github.com/covid19india/api

SEIR Model with social distancing from: https://aalexan3.math.ncsu.edu/articles/covid_control.pdf

Here, I will try to fit the SEIR(Susceptible, Exposed, Infected, Recovered) model with social distancing incorporated to Maharashtra Covid-19 data. u denotes level of social distancing, u=0 means no social distancing and u=1 means full lockdown.
In a closed population (that is, assuming no births or deaths), the SEIR model with social distancing incorporated is:

\[ \begin{aligned} \frac{dS}{dt} & = -\frac{(1-u)\beta IS}{N}\\ \\ \frac{dE}{dt} & = \frac{(1-u)\beta IS}{N} - \kappa E\\ \\ \frac{dI}{dt} & = \kappa E - \gamma I\\ \\ \frac{dR}{dt} & = \gamma I \end{aligned} \]

Express the differential equations of SEIR model with social distancing as an R function.Full lockdown is impossible, as some people will have to be out for maintaining essential services. By associating cost with social distancing optimal value of u found was 0.6031.
Source: https://aalexan3.math.ncsu.edu/articles/covid_control.pdf

SEIR <- function(time, state, parameters) {
  par <- as.list(c(state, parameters))
  with(par, {
    dS <- -(1-0.6031)*beta * I * S/N
    dE <- (1-0.6031)*beta * I * S/N - kappa * E
    dI <- kappa * E - gamma * I
    dR <- gamma * I
    list(c(dS, dE, dI, dR))
  })
}

Nation wide lockdown started on 25th March. So, consider the cumulative infected/confirmed cases from 25th March 2020 till 30th April 2020 for finding optimal values of model paramters

Infected <- mh_dat %>% filter(Date >= dmy("25-Mar-20"), Date <= dmy("30-Apr-20")) %>% 
  pull(Cum_Confirmed)

Create an incrementing Day vector the same length as Infected. Initialize N with a population of Maharashtra. Initially, everyone is in susceptible class, so initialize Susceptible with S = N - cumulative confirmed cases on 25th march(First day of data), Infected (I) with cumulative Infected cases on 25th March, Exposed (E) with twice of Infected cases, recovered R = 1 from the data.

Day <- 1:(length(Infected))
N <- 114200000
init <- c(S = N - Infected[1], E=Infected[1]*2, I = Infected[1], R = 1)

Fit the SEIR model to our data by finding the values for β, γ, κ. κ is inverse of incubation period, median incubation peroid is around 5 days, so set κ to be 0.2. Based on recent news recovery rate of India is around 34% and is improving, for Maharashtra it’s low but improving. So, set γ to be around that, take it as 40%. For Maharashtra, daily cases are very high, so transmission coefficient β must be high. Using trial and error a good β can be 1.97.

Opt_par <- setNames(c(1.97, 0.2, 0.4), c("beta", "kappa", "gamma"))
